## 背景

本科毕业后，我想提高我的计算机科学水平，想以美国顶尖大学的标准重新学习计算机科学，选择跟随[TeachYourselfCS](https://teachyourselfcs.com/)的指导坚持学习。

第一门课程是《计算机程序构造与解释》。

## 下载

```sh
$ git clone https://github.com/wuxueqian14/CS61A-Spring2021.git
```

## 使用方法

- 打开你想要查找的`lab`文件夹。 
- 打开你想要查找的`homework`文件夹.
- 打开你想要查找的 `project`文件夹.

查看`xx.py`代码文件。
